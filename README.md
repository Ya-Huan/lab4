sizeof函數可用來取得變數的位元組大小(所配置的記憶體大小)。
char型態大小為1byte;
int型態大小為4bytes;
float型態大小為4bytes;
double型態大小為8bytes。

pointer是專門用來儲存變數的記憶體位置。
每宣告一個pointer，就會配置一塊記憶體空間，
(電腦為64位元情形下)其大小為8bytes。
